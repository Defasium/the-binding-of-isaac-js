Quitter l'outil
  •  
  • Cliquez ici si rien ne s'affiche
  •  
  • Bienvenue dans l'outil créateur de salle !
  • INSTRUCTIONS :
  • CLIC GAUCHE sur la liste pour choisir l'élément
  • CLIC GAUCHE sur la grille pour poser l'élément
  • CLIC DROIT sur la grille pour effacer les objets
  • BOUTONS :
  • Appuyez sur TEST pour visualiser et essayer votre création
  • Appuyez sur EDIT pour revenir dans l'outil créateur
  • Appuyez sur RESET pour réinitialiser la grille
  • EXPORTER :
  • SAVE télécharge votre création sous forme de fichier texte
  • La zone de texte de droite détermine le nom du fichier
  • IMPORTER :
  • Copiez-collez le code complet d'une sauvegarde dans la zone de texte de gauche et appuyez sur READ pour importer le code dans l'éditeur
  • Appuyez sur CLEAR pour vider la zone de texte
